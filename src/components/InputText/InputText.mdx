# InputText

`InputText` is a wrapper around
[Gutenberg TextControl](https://github.com/WordPress/gutenberg/tree/master/packages/components/src/text-control)
component and
[Gutenberg TextareaControl](https://github.com/WordPress/gutenberg/tree/master/packages/components/src/textarea-control)
component.

`InputText` is used as a text input. It is used to accept a users text
interactions.

## Usage

```ts
import { InputText } from "@fmb/components/InputText";
```

<Playground>
  {() => {
    const [value, setValue] = useState("");
    return (
      <div>
        <InputText
          placeholder="Who is your favorite basketball team?"
          onChange={handleChange}
        />
        Your favorite team is: {value}
      </div>
    );
    function handleChange(newVal) {
      setValue(newVal);
    }
  }}
</Playground>

## Multiline

To have the `InputText` render as a textarea instead of an input, use the
`multiline` prop.

</Playground>
